# Plant Disease Dataset

This folder contains datasets used for training and testing the AI Based Plant Disease Detection model.

Dataset Source:
- PlantVillage Dataset
- Custom Leaf Image Dataset

Purpose:
- Disease Classification
- Healthy Plant Detection
- Model Training
- Model Evaluation

Supported Crops:
- Tomato
- Potato
- Pepper
- Corn
- Apple
